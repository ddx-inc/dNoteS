dNoteS - beautiful notes on your desktop.
Version 0.2

**Features:**
+ Pinning to screen
+ Always ahead
+ Various themes

**Hotkeys:**
Ctrl + N - new note
Ctrl + (+) - pin note
Ctrl + (-) - unpin note
Ctrl + Q - close note
F1 - about dNoteS
RMB - меню

Written in Python 3.8 (tkinter)
